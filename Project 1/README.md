# Cafe Sales Data Cleaning

## Description
This project demonstrates data cleaning and preprocessing of a cafe sales dataset.
The goal was to prepare raw data for further analysis.

## Problems in the raw data
- Missing values
- Duplicate rows
- Incorrect data types
- Inconsistent formatting
- Currency symbols in numeric columns

## Cleaning steps
- Removed duplicate records
- Handled missing values
- Converted columns to correct data types
- Cleaned numeric columns from text and symbols
- Standardized column names

## Tools used
- Python
- Pandas
- Excel

## Result
Clean and structured dataset ready for data analysis and visualization.
